Basic api setup with node and postgres
connection to db info contains in .env

[![node][node]][node-url]

If you have no .env file in project
root directory make one with this variables
 - PGUSER
 - PGHOST
 - PGPASSWORD
 - PGDATABASE
 - PGPORT
 - SECRET
 - PORT
 - HOST
